Types in Js

There are two types in Js: Primitive and Reference.

Primitive store the value itself while reference is a pointer to memory location.

Therefore assigning primitive var to another creates a copy while in case of reference type only reference is copied.

Primitive types

                        // There are 5 primitive types
                        // number
                        var count = 10;
                        var cost = 1.15;

                        // string
                        var name = 'Ali';
                        var selection = 'z';

                        // boolean
                        var found = true;

                        // null
                        var object = null;

                        // undefined
                        var flag = undefined;
                        var ref; // assigned undefined automatically

                        console.log('These are the 5 primitive types in JS');
                        console.log(count);
                        console.log(name);
                        console.log(found);

                        console.log(object);
                        console.log(flag)

                        // To check type use typeof
                        console.log(typeof flag);
                        console.log(typeof cost);

Despite the fact that they have methods, primitive values themselves are not objects. JavaScript makes them look like objects to provide a consistent experience in the language. 

                        // methods on primitive types
                        console.log(cost.toFixed(1)); // prints 1.1
                        console.log(name.toUpperCase());  // prints ALI
