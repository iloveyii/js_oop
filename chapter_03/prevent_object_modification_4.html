Prevent Object Modification

There are two ways of creating objects in Js, using the Object constructor and using the Object literal.

When we create a property of an object for the fist time, the JS engine calls [Put] method, but when we update it with another value it calls [Set] method. The Put method makes the hasOwnProperty true. The hasOwnProperty is different than prototype properties.

Detecting properties

To detect if a property is in an object use the 'in' operator. The in operator checks both own and prototype properties. To see own property use hasOwnProperty method instead.

To remove an object property use 'delete'.
